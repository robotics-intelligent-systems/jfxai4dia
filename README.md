## Roadmap stages

CEASIOMpy is an conceptual aircraft design environment. Simple Modularization Process applied to Ship Design. CalculiX Extras. ADAO - SALOME module for Data Assimilation and Optimization. FAST-OAD is an open source framework for rapid Overall Aircraft Design. ShipHullGAN is an AI‑Driven Ship Hull Generation. C-ShipGen is a Conditional Guided Diffusion Model for Parametric Ship Hull Design. CQAsk utilizes LLMs to generate CAD models. GPkit is a Python package for defining and manipulating geometric programming models, abstracting away the backend solver. OpenSBLI is an open-source code-generation system for compressible fluid dynamics (CFD). The AI 3D Generator is a FreeCAD plugin that integrates OpenAI's powerful language models to generate and execute 3D models through AI-driven code generation. The NVIDIA (DALI) is a GPU-accelerated library for data loading and pre-processing to accelerate deep learning applications. NVIDIA Warp is a Python framework for writing high-performance simulation. CFL3D is a structured-grid, cell-centered, upwind-biased, Reynolds-averaged Navier-Stokes (RANS) code. ModelFLOWs is a research group specializing in data analysis, reduced order modeling, machine learning, artificial intelligence and computational fluid dynamics (CFD). The Multidisciplinary-design Adaptation and Sensitivity Toolkit (MAST) is a sensitivity-enabled, multiphysics FEA tool developed to support computational design and engineering analysis. DrivAerNet++ covers all conventional car designs. BRL-CAD is a powerful cross-platform combinatorial solid modeling system. CadQuery is an intuitive module for building parametric 3D CAD models. CADCloud is a cloud based sharing infrastructure dedicates to CAD engineers. The FreeCAD MCP (Model Control Protocol) provides a simplified interface for interacting with FreeCAD through a server-client architecture. Mechatronic is a Workbench for FreeCAD. mshr is the mesh generation component of FEniCS

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.

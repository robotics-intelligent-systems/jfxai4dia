# Artificial Intelligence for Modular Industrial Design

IfcOpenShell is the open source IFC toolkit and geometry engines. CAD-Editor a Locate-then-Infill Framework with Automated Training Data Synthesis for Text-Based CAD Editing. Ptera Software is a fast, easy-to-use, and open-source software package for analyzing flapping-wing flight. CEASIOMpy is an conceptual aircraft design environment. JSCAD is a set of modular, browser and command line tools for creating parametric 2D and 3D designs with JavaScript code. ADAO - SALOME module for Data Assimilation and Optimization. FAST-OAD is an open source framework for rapid Overall Aircraft Design. ShipHullGAN is an AI‑Driven Ship Hull Generation. C-ShipGen is a Conditional Guided Diffusion Model for Parametric Ship Hull Design. CQAsk utilizes LLMs to generate CAD models. Mesh Viewer is a simple STL/OBJ CAD geometry and surface mesh viewer. GPkit is a Python package for defining and manipulating geometric programming models. OpenSBLI is an open-source code-generation system for compressible fluid dynamics (CFD). The AI 3D Generator is a FreeCAD plugin. QPlane is a fixed wing flight simulation environment. VGL is an open source and cross platform aeromechanics in .NET. Python Flight Mechanics Engine. CFL3D is a structured-grid, cell-centered, upwind-biased, Reynolds-averaged Navier-Stokes (RANS) code. ModelFLOWs is an open-source modelling, simulation, reduced-order modelling, and data-driven tools for engineering and biomedical applications. The Multidisciplinary-design Adaptation and Sensitivity Toolkit (MAST) is a sensitivity-enabled, multiphysics FEA tool. DrivAerNet++ covers all conventional car designs. BRL-CAD is a powerful cross-platform combinatorial solid modeling system. CadQuery is an intuitive module for building parametric 3D CAD models. The FreeCAD MCP (Model Control Protocol) provides a simplified interface for interacting with FreeCAD through a server-client architecture. Mechatronic is a Workbench for FreeCAD. mshr is the mesh generation component of FEniCS

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
